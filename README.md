# ChainSubscription-Hub

Stellar Soroban bootcamp project — Counter-with-owner example.

How to build & deploy
1. Open https://soroban.studio/ and clone this repo.
2. Open `contracts/hello-world` and press Build.
3. Connect Freighter (Testnet) and Deploy.
4. Invoke `init`, `inc`, `get` to interact.

Contract files
- `contracts/hello-world/src/lib.rs` — Counter-with-owner Smart Contract
